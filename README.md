# 社团信息管理系统 java375

## 项目概述

社团信息管理系统是一个基于B/S架构和MVC设计模式的Web应用。该系统实现了前后端分离，前端使用Vue框架，后端采用Java语言和SpringBoot框架，旨在为学校或社区的社团提供一个便捷、高效的信息管理平台。

## 技术栈

- **系统环境**：Windows/Mac
- **开发环境**：
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术框架**：
  - Java
  - MySQL
  - SpringBoot
  - Mybatis
  - Vue
